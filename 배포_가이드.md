# 수심달 랜딩페이지 배포 가이드

## 1단계 — 의존성 설치 & 로컬 실행 확인

터미널을 열고 이 폴더로 이동한 뒤 아래 명령어를 실행하세요.

```bash
cd susimdal-landing
npm install
npm run dev
```

브라우저에서 http://localhost:3000 을 열어 랜딩페이지가 잘 보이면 OK!

---

## 2단계 — GitHub 리포지토리 생성 & 푸시

### 방법 A: GitHub 웹사이트에서 생성 (추천)

1. https://github.com/new 접속
2. Repository name: `susimdal-landing`
3. Public / Private 선택 후 **Create repository**
4. 아래 명령어 실행:

```bash
git init
git add .
git commit -m "Initial commit: 수심달 랜딩페이지"
git branch -M main
git remote add origin https://github.com/susimdal-edu/susimdal-landing.git
git push -u origin main
```

> Git 처음 사용 시 push할 때 GitHub 로그인 창이 뜹니다. 구글 계정으로 로그인하면 됩니다.

---

## 3단계 — Vercel 배포

### 방법 A: Vercel 웹 대시보드 (가장 쉬움)

1. https://vercel.com/dashboard 접속
2. **Add New → Project** 클릭
3. GitHub 계정 연동 → `susimdal-landing` 리포 선택
4. **Deploy** 클릭

→ 약 1분 후 `https://susimdal-landing.vercel.app` 주소로 배포 완료!

### 방법 B: 터미널에서 Vercel CLI 사용

```bash
npx vercel
```
(처음 실행 시 브라우저에서 Vercel 로그인 → 자동 배포)

---

## 이후 수정사항 반영 방법

코드 수정 후 아래만 실행하면 Vercel이 자동으로 재배포합니다:

```bash
git add .
git commit -m "업데이트 내용"
git push
```
